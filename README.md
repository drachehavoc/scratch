# O que são garatujas?

Garatujas é o nome dado às expressões visuais de uma fase inicial do grafismo das crianças. Esses desenhos são muito importantes para que elas busquem representar sua realidade.

As garatujas expressam sentimentos e dão sentido ao mundo da criança. Elas podem ser de diferentes tipos e representam os primeiros passos na direção de um controle motor e do desenvolvimento criativo das crianças.
